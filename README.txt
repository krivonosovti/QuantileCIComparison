Курсовая работа на тему "Сравнение методов расчета доверительных интервалов для 90% и 99% квантилей с использованием синтетических данных и данных из Kaggle в контексте производительности и задержек."
Методы представленные к сравнению:  Bootstrap, Delta method, Simple (Naive) Method
Задача сравнения производительности и latency

Файлы и директории:
  Course_Work/: Директория, где находится PDF-файл и файлы LaTeX, используемые для создания отчета.
  Data/: Директория с синтезированными выборками данных. (хаб добавил ограничение на кол-во файлов, приложил скрин  Data.png)
  DataFactory.py: Программа для создания синтетических выборок данных.
  Naive.ipynb: Нормальный вариант для наивного метода расчета доверительных интервалов.  // нужно написать функцию и заменить в  SimpleMethod.py.
  BootstrapMethod.py: Программа для расчета доверительных интервалов с использованием метода бутстрепа.
  DeltaMethod.py: Программа для расчета доверительных интервалов с использованием дельта метода. Обратите внимание, что исходный вариант этого метода не является правильным.

  Delta.py и Delta.ipynb: Файлы для попыток разобраться с дельта методом.

  analysis.py: Программа, которая собирает данные из ResData и проводит их сравнение, включая построение графиков.
  ResData/: Директория, содержащая результаты работы программы analysis.py.
